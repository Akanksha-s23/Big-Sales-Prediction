# BIG SALES PREDICTION USING RANDON FOREST REGRESSOR IN MACHINE LEARNING

# Overview

This project aims to predict sales of products across different retail outlets using machine learning. By analyzing historical sales data, the model             identifies patterns and relationships between product attributes and their sales performance.

# Objective

To build a predictive model that accurately estimates Item Outlet Sales based on product and outlet features.

# Dataset
    1) Source: YBI Foundation Dataset
    2) Contains 14,000+ records with features like:
            Item Weight
            Item Fat Content
            Item Visibility
            Item Type
            Item MRP
            Outlet Type, Size, and Location

# Technologies Used
   1) Python
   2) Pandas, NumPy
   3) Seaborn (for visualization)
   4) Scikit-learn (ML model & preprocessing)

# Steps Involved
    1) Data Loading and Exploration
    2) Handling Missing Values
    3) Data Cleaning and Encoding
    4) Feature Scaling using StandardScaler
    5) Train-Test Split
    6) Model Training using Random Forest Regressor
    7) Prediction and Evaluation

# Model Used
    Random Forest Regressor : Chosen for its robustness and ability to handle non-linear relationships in data.  

# Key Insights
       1) Item MRP plays a significant role in determining sales.
       2) Outlet type and location impact product performance.
       3) Data preprocessing improves model efficiency and accuracy.

# Future Improvements
      1) Hyperparameter tuning for better accuracy.
      2) Trying advanced models like XGBoost.
      3) Building a web app for real-time prediction.

# Conclusion
    The project successfully demonstrates how machine learning can be used to predict sales and support decision-making in the retail industry.
